ABOUT ME

My name is Richard Breau and I graduated from Eastern College in 2021 with a diploma in Advanced Systems Management and Cybersecurity. Since then I have completed two certifications from CompTIA in A+, as well as Security+, and also had completed Microsofts Certified Security Operations Analyst Associate, Microsoft Cybersecurity Architect Expert, Microsoft Azure Administration Associate. This has been the start of my path towards my main goal of being part of the security solutions using AI and all the potential possibilities it holds. I started with some simple programs such as guess the number, a tip calculator, and built up the difficulties by creating a golf hanicap calculator, and recreating projects such as the old pong game, and a count down clock towards my goals.

Using various tools I have constructed this personal website that I have online for all to see, but with the goal of creating one using absolutely no other template but that which I make, using the styling and javascript from my own typing. The purpose of this is to show myself that I am able to put in code what I see in my mind and not limited by my skills of what I can do with another's template. This might not seem that special, but for someone like me who didn't start learning this until 2020, it is proof of what Walt Disney said It's kind of fun to do the impossible so with the confidence this give me, my next challenge is to take on the impossible.
